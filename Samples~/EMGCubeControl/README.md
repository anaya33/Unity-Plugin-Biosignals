# EMG Cube Control Sample

## Setup

1. Create a new scene
2. Add a Cube to the scene
3. Create an empty GameObject called "BrainFlowManager"
4. Add the `BrainFlowManager` component to it
5. Add the `BiosignalInput` component to it
6. Add the `EMGCubeController` component to the Cube
7. Assign references in the Inspector

## Usage

1. Press Play
2. Flex your muscle
3. Watch the cube move up!

## Configuration

- **Max Height**: How high the cube can go
- **Responsiveness**: How fast the cube reacts
- **Change Color On Flex**: Visual feedback when flexing
